# Plummet

A mace PvP training mod for Minecraft. Spawn practice bots and run drills in singleplayer.

Built for Fabric 1.21.11.

## Status

Early. The command skeleton works and spawns a basic practice dummy. The real bot AI is in progress.

## Build

Requires JDK 21.

```
./gradlew runClient
```

Then open a world and run `/plummet bot spawn`.

## Versions

- Minecraft 1.21.11
- Yarn 1.21.11+build.6
- Fabric Loader 0.19.3
- Fabric API 0.141.4+1.21.11
